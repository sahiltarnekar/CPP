# Dynamic Memory Allocation - Linked List CRUD Operations

This C++ project demonstrates dynamic memory allocation through the implementation of Create, Read, Update, and Delete (CRUD) operations on a Singly Linked List.

## Features Implemented

- **Class Structure**: A `DynamicMemoryAllocation` class that handles the linked list operations and memory management.
- **Append**: Adds a new node to the end of the list using `append()`.
- **Display**: Iterates through the list and prints all elements using `display()`.
- **Insert at Beginning**: Adds a new node to the start of the list using `insert_at_beginning()`.
- **Search**: Looks for a specific value in the list using `Search()`.
- **Delete Node**: Removes a node containing a specific value and frees its memory using `Delete_node()`.
- **Reverse**: Reverses the order of nodes in the linked list using `reverse()`.

## How to Compile and Run

To compile and run this project, open your terminal/command prompt, navigate to the folder, and use `g++`:

```bash
g++ main.cpp -o dynamic_memory
./dynamic_memory
```

*(On Windows, run the compiled file with `.\dynamic_memory.exe`)*
