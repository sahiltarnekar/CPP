# Memory Calculate - Student Management System

This C++ project implements a simple Student Management System utilizing `std::vector` and class templates.

## Features Implemented

- **Class Template (`MemoryCalculate`)**: A template class representing a student with an ID and Name.
- **Add Student**: Adds a new student to the system using `push_back()`.
- **Display Students**: Iterates through the vector to show all student records.
- **Remove Student**: Finds a student by their ID and removes them from the vector using `erase()`.
- **Search Student**: Finds and displays a specific student based on their ID.

## How to Compile and Run

To compile and run this project, open your terminal/command prompt, navigate to the folder, and use `g++`:

```bash
g++ main.cpp -o student_management
./student_management
```

*(On Windows, run the compiled file with `.\student_management.exe`)*
