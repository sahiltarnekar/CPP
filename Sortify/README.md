# Sortify - DSA with C++

This project contains solutions to core array operations and sorting techniques in C++.

## Missions Included

1. **Mission 1: Remove the Twins** (`task1_remove_duplicates.cpp`)
   - Removes duplicate elements from an array and displays the unique values.

2. **Mission 2: Topper Finder** (`task2_topper_finder.cpp`)
   - Finds the largest and second-largest numbers in an array. Handles duplicates gracefully.

3. **Mission 3: Sort Squad** (`task3_sort_squad.cpp`)
   - Sorts an array in both ascending and descending order using Bubble Sort.

4. **Mission 4: Count Values** (`task4_count_values.cpp`)
   - Counts and prints the total number of positive and negative numbers in the given array.

5. **Mission 5: Magic Sorter** (`task5_magic_sorter.cpp`)
   - Separates even and odd numbers, sorts even numbers in ascending order, odd numbers in descending order, and merges them back together.

## How to Compile and Run

For any given task, use `g++` to compile and then run the generated executable. For example, for task 1:

```bash
g++ task1_remove_duplicates.cpp -o task1
./task1
```

*(On Windows, the compiled file will be `task1.exe` so you can run it with `.\task1.exe`)*
